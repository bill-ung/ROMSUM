ROMSUM is a command-line utility that calculates a straight checksum value, for any given image. ROMSUM was written primarily for the pinball world, and the utility is designed to recognize known EPROM sizes. It also includes the ability to calculate the checksum for each 0x1000 page, which is helpful when working with Williams System 11{A,B,C} ROMs.

ROMSUM Version 2.5 (ROMSUM.exe) is a 16-bit binary that will run under older versions of MS-DOS, or from a 16 or 32-bit Windows Command Prompt. Limited to 8.3 style filenames.

ROMSUM Version 3.1 (ROMSUM32.exe) is a 32-bit binary that must be run from a 32 or 64-bit Windows Command Prompt. Supports long filenames.
