The Pinball Rom Identifier is an extension of the ROMSUM utility. You may select any ROM file you may have and upload it here. The tool will attempt to match the file against known ROM images, based on file size and checksum values. If a match is found, the game information and details for the specific EPROM chip are displayed.

If your file does not match, send an E-Mail and I'll see if I can identify your file manually. Once identified, you may also wish to submit the file to the IPDB and/or PinMAME as a contribution to the pinball hobby.

This is an experimental tool, written by Bill Ung using Linux (CentOS), Apache (http), MySQL (MariaDB) and PHP, plus HTML/JavaScript
Database information was generated from personally collected ROM files, the Internet Pinball Database (IPDB), and the PinMAME project.

If you're looking for the older ROMSUM command-line utilities, click here.

If you have any feedback regarding the Pinball Rom Identifier utility, feel free to send E-Mail.
